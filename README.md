## Why ZapFile.ai?

ZapFile.ai is designed for fast, private, and simple file sharing:

- Direct browser-to-browser transfer (WebRTC)  
- One-time share links and QR codes  
- Unlimited file size  
- No server uploads or history stored  

---

## How It Works

1. Open https://zapfile.ai  
2. Select the file you want to send  
3. A secure WebRTC connection is created  
4. Share the link or QR code with the recipient  
5. Transfer completes directly to the other device
